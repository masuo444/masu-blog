# 273:進化するWeb開発とボランティア活動の融合

7/1

今日も一日、AIとの対話に時間を費やしました。単なる作業というよりは、まるで新しい言語を学ぶかのように、AIとのコミュニケーションを深めることに集中していました。

特に今日は、ほぼフルタイムで「Vibes Coding」に取り組んでいましたね。

**AIへの飽くなき探求と「クロードコード」の限界**
実は、昨晩は徹夜をしました。

Vibes Codingを少しでも早く習得し、自由自在に使いこなせるようになりたいという強い思いから、睡眠時間を削ってまで没頭していました。

現在使っているのは「クロードコード」というサービスです。最初は月額20ドルのプランで試していたのですが、驚くほどあっという間に上限に達してしまい、正直物足りなさを感じました。

　　

そのため、わずか1日後にはすぐに100ドルのプランにアップグレードしました。200ドルの最上位プランもありますが、そこまでは必要ないだろうと判断したのですが、この100ドルプランでも、連日使い続けていると普通に上限にぶつかってしまいます。

どうやら5時間に1回リセットされる仕組みのようで、AIを使った作業中に「あと2時間使えません」といった表示が出ることがあります。

そんな時、僕はその「待機時間」を有効活用しています。AIが使えない2時間は、お城でのボランティア活動に時間を充てるんです。

そして、またAIが使える時間になれば、すぐに戻って開発作業を再開します。このサイクルが、僕の現在の働き方になっています。

**AIが拓くWebサービス開発の可能性**
今、AIに力を借りて取り組んでいるのは、主にウェブサイトの構築です。

自身のパフォーマンスブランドの公式サイトをはじめ、コンサルをしている間に、こんなのが欲しい！

という想いを形にするためのホームページ**もいくつか手掛けています。

もしかしたら、これが将来的に大きなビジネスに発展する可能性も秘めていると感じています。

そして、僕自身もAIを活用した新しいウェブ

サービスを開発しています。

このウェブサービスが、もしかしたらマネタイズ可能なビジネスになり得るのではないかという可能性に気づき、現在、Vibes Codingを使ってその開発を進めているところです。

**知識ゼロから創造する喜び**

かつてはエンジニアやプログラマーといった専門知識を持つ人々にしか不可能だったことが、今は違います。プログラミングの知識が一切ない僕でも、頭の中のイメージや創造力さえあれば、それをAIを使って具体的な形にできる時代になったんです。

Vibes Codingが登場したのはごく最近のことなので、本当に良い時期にこの技術に集中して取り組めていると感じています。

ボランティア活動で地域に貢献しつつ、その合間にAIのスキルアップを図る。

これは非常にありがたいことです。お城のオーナーからも、ボランティア活動は時間を強制されるものではなく、隙間時間に行ってくれれば良いという寛大な配慮をいただいているので、短時間で集中してボランティアの成果を出し、それ以外の時間をすべてAIのトレーニングに充てています。

この働き方は、僕にとって非常に居心地が良く、大きなやりがいを感じさせてくれます。本当に、できることが格段に増えました。

わずか3、4日前からVibes Codingに没頭していますが、その習熟度には目覚ましいものがあります。

以前は一つの画面で一つのサイトを構築するのが精一杯でしたが、今では同時に3つ、4つのサイトを並行して作成できています。たった数日で、効率が3倍から4倍にもなった計算です。

本来なら何十万、何百万円とかかるウェブサイトの構築が、僕一人にかかる人件費と、API利用料やサーバー代といった最小限の費用を合わせても、わずか数万円（僕の人件費を含めなければ）で済んでしまう。こんなにも便利な時代が来たことに、ただただ驚くばかりです。

人とほとんど会わずに作業に集中できること、そして、普段はホテル暮らし（ドミトリーが多いので誰かが常にいる）ですが、ここお城では個室を与えられているため、誰に邪魔されることもなく、じっくりと作業に没頭できる環境も大きいです。

AIの学習や開発にこれほど適した場所は他にはないのではないでしょうか。

これからも、この新たな働き方を追求し、AIと共に進化し続けたいと思います。
