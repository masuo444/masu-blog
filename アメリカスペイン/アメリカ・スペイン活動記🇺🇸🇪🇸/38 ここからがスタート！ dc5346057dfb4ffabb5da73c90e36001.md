# 38:ここからがスタート！

2024/07/28

さて、

約40日間のアメリカ・スペイン遠征も終わり、

本命のアイルランドでの活動がスタートします。

もともと、7月28日にアイルランドに行くことは確定しており、その後に、アメリカやスペインのスケジュールを入れたのが正しいです。

とはいえ、

最初は20日くらいで考えていたのですが、

GPSランナーの志水さんの

ペルーチャレンジの帰国日に合わせた結果、

結局40日間の遠征になったわけで、

毎日毎日、色々な発見があったので、

ある意味必然だったのかもしれません。

たった40日間ですが、

本当にいろんなことがありました。

たくさんの出会いや、

一流のエンターテイメントに触れたり、

現地の方の家に泊まり、

良いことも悪いこともあったり、

飛行機乗れなくて、

急遽、夜行バスで向かったり、　

「服着てることが不快」と言われたり、

日本で普通に生きていたら

なかなか出来ない経験をすることができました。

僕にとって、この経験は資産であり、

この記事は僕自身がこの日々を

忘れないように書き記したものですが、

これを皆さんにも共有することで、

2024年現在の海外の事情や、

僕の活動を少しでも面白がってもらえたら、

嬉しいです。

さて、ここからは、アイルランド🇮🇪へ行きます。

このアイルランドで、枡ブランド「FOMUS」を拡げるために、僕は約一年ほど、アイルランドを拠点に置いて活動をします。

とはいえ、

ずっとアイルランドにいるというよりは、

ヨーロッパを周りながら

活動するという感じです。

日本からヨーロッパへ行くよりも、

アイルランドから他のヨーロッパへ行く方が

圧倒的に、早いので。

8月〜10月は基本的には

アイルランドで活動予定です。

9月は、メドックマラソンの期間だけは、

ボルドーへ移動します。

基本的な活動としては、

最初の3ヶ月は

アイルランドにいる人達との繋がりを作ること。

実は、アイルランドの語学学校に通います。

理由はいくつかあります。

1、英語力の強化

━━━━━━━━━━━━━━

まず、英語力の強化。

アイルランドは、英語ですが、

少し訛っているそうで、

ここで、しっかりと聞き取る能力を身につけることができれば、他の国でも通用すると考えています。

また、世界で仕事をする上で、「英語」は、

現状、コミュニケーションをとる上で、

必ず必要なので

30歳になるまでには、しっかりと身に付けます。

そして、30歳以降は、英語を使って、

海外でもビジネスを展開していきます。　

2、「ヨーロッパの繋がりを作ること」

━━━━━━━━━━━━━━

語学学校には、

ヨーロッパ各地から人が集まります。

つまり、ここで繋がりを作ることで、

10月以降の、

他の国への足掛かりを作りやすくなります。

ぶっちゃけ、フィリピンの3ヶ月は、

フィリピンでのFOMUSブランドの広がりもできましたが、

その学校内の横のつながりを作ることもできたんです。

彼らは、

いまも僕のSNSでの発信を見てくれていて

たまに連絡も取り合う仲です。

来年以降、東南アジアに行く時は、

彼らと待ち合わせして行くのも良いですし、

彼らが日本に来る時は、

FOMUSをふんだんに活用して、

交流の生まれる機会を創出したいと考えています。

つまり、この3ヶ月間で、

同じ学校に通ってる人達に、

「まっすーと一緒にいると楽しい」と

思ってもらうことが重要です。

それが、その後に繋がってくるということです。

3、ホームステイで、文化を知る

━━━━━━━━━━━━━━

これから2ヶ月間は、

ダブリンでホームステイです。

語学学校に通うことで、

ホームステイか、学校の寮を選べるのですが、

今回はホームステイ。

それは現地に根付いている方との交流が、

最も大切だからです。

ある程度の英語力は必要ですが、

このコミュニケーションが、FOMUSや僕を、

アイルランドに根付かせるために

必要なことだと思っています。

他にも、色々な理由がありますが、

アイルランドで語学学校に通います。

アイルランドには、

ギネスビールや、

アイリッシュウイスキー、

アイリッシュ民謡、

など、FOMUSと

掛け合わせたいコンテンツが

沢山あります。

特に、アイリッシュミュージックは、

FOMUSのイベント時に使う音楽や、

キャラクターKUKUの曲としても活用予定で

著作権が切れた名曲を見つけてきます！

そして、

アイルランドも、おそらく村社会であるので、

しっかりと村人として認めてもらい、

様々なコラボを通して、

FOMUSを拡げていきます。

そんな感じで、

アイルランドでの活動になります。

ここからがメインです！！笑

正直、予想外のことも沢山起きるでしょう。

貴重な20代のラストを、

アイルランドでの活動に費やすことが

正しいのかどうかも、誰にも分かりません。

それでも、30代は、

20代で築いてきた能力やスキルを存分に活かして

活動をしていくために、

今できることを全力でやっていきます。

そして、日本でも進めている、

FOMUSが関わるプロジェクトにも

良い影響が出るように、

うまく連動させていきます！！

というわけで、

マドリードからダブリンのフライト中に

今の想いを書きました。

いまは、期待と不安が混ざってます。

それすらも、楽しんでいこうと思います！！

ここまで読んでいただきありがとうございました！！
