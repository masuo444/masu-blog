# 【3日目】日本のコンテンツは強い

マレーシア3日目。

午前中は

クアラルンプール在住のモデルさんと撮影がありました。

昨日の夜に、この枡フォトは決まったので、

本当に来てくれるか不安もありましたが

無事に撮影することができました！

相手はイラインさんという方で、

モデル活動をされている20歳。

今回は同伴でお母さんも一緒に。

お母さんの親戚は、香港で日本語を教えているそう。

当たり前ですが、

2人とも日本語は話せないので

やり取りは全て英語です。

枡フォトの良いところは、

あまり難しいワードを使わなくても可能なところ。

撮影しない移動中は、

英語で会話をしながらコミュニケーションをとります。

カメラマンの上手い下手は、

ここに出ると思っていて、撮影していない時の

会話が盛り上がっていれば自然と良い写真が撮れます。

#出会って10秒以内に相手の緊張を解く

#ホテルマン時代に鍛えられた

そんなこんなで、

今日も枡アクセサリーを身に付けて写真撮影。

撮影時間は30分ほどなのですが、

その間だけで、モデルの子と、お母さんまで、

「このアクセサリー欲しいから買いたい！」と言っていただきました。

やはり、枡という概念がそもそも無いからこそ、

「日本の伝統工芸」をファッションとして身に付けられるのに魅力があるのでしょうか？？

このあたりはもう少し、理由を揃えていきたいですね。

**日本のコンテンツは強い**

ーーーーーーーーーー

さて、話は変わりますが、

クアラルンプールの紀伊國屋へ行ってきました。

日本のアニメや漫画は、

世界で日本が勝てるコンテンツなので、

そこの勉強もしっかりと。

クアラルンプールの様々な人種がいるからなのか、

英語版、中国語版、日本版が、

それぞれブースに分かれて置かれていました。

面白かったのは、

それぞれの翻訳によって金額がバラバラということ。

例えば、

スパイファミリーは

英語版　47.65RM

中国　20.9RM

日本語　30.36RM

推しの子も

英語版は64.22RM

中国は26.9RM

日本語37.48RM

1RM→31円〈2023/09/08現在〉

圧倒的に英語版が高いんです。

推しの子なんて、

日本語版が1200円で

英語版が2000円です。

輸送コストや

紙の原材料費なども関係しているのでしょうが、

なかなかのお値段ですよね。

なんでこんなに金額が違うんでしょうね？？笑

紀伊國屋ということもあり、

日本のマンガ・小説系はとにかく豊富でした。

いまリアルタイムで

放映されているアニメの原作コミックの棚などもあったのですが、

世界中でアニメが観れる時代だからこその

配置だなぁと思いました。

ちなみに、グッズもまとめて売っていました。

**マレーシアは飯に困らない国**

ーーーーーーーーーー

さて、最後に雑談ですが、

クアラルンプールに関しては、

一切、ご飯には困らないです笑

僕はKLCCという都心部に拠点を置いてるのですが

近くには巨大なショッピングモールが複数あり、

そこに行けば、日本食もタイ料理もマレーシア料理もドバイ料理もハラール料理も、だいたい揃ってます。

しかもだいたい600円以内で食べれます。

今日の僕の夜ご飯は、

ショッピングモールの中にあるフードコートの

タイ料理屋のチャーハンでしたが、450円。

めちゃ安くて、美味しい！

非常にクアラルンプールおすすめです！！
