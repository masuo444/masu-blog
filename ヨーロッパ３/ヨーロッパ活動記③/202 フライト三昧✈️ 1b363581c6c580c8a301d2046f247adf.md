# 202:フライト三昧✈️

2025/03/09

さて、ドバイからパリへ移動するのですが、

交通費を節約するために、

国を経由しながら、向かいました。

ドバイ➡︎アンタルヤ

アンタルヤ➡︎ロンドン

ロンドン➡︎パリ

です。

1日に3回のフライトです。

前回の記事でも書きましたが、

ドバイ空港は90分ほどしかラウンジが使えず。

それでも、余裕でラウンジを楽しめましたし、

スタッフも優しいです。

ここでは、久しぶりにビールも飲めました🍺

1人で晩酌をすることは、あまりありませんが、

ラウンジは1人でもガッツリ飲みます笑

アンタルヤには朝の7時ごろに到着。

ここでもラウンジに入って朝食。

※1枚も写真を撮ってない

クオリティはあまり高くなかったですが、

休むのには問題なかったです。

そして、ロンドンへ。

ロンドンのガードウィックという空港に到着したわけですが、ここはラウンジの料理も美味しかったです。

ビールも飲み放題で、

非常に居心地が良かったです。

![IMG_8777.jpeg](202%20%E3%83%95%E3%83%A9%E3%82%A4%E3%83%88%E4%B8%89%E6%98%A7%E2%9C%88%EF%B8%8F/IMG_8777.jpeg)

そして、パリへと到着します。

正直、乗り継ぎで毎回チェックインをして、

手荷物検査して、入国審査して、

という流れは面倒ですが、

プライオリティパスを持っているので、

ラウンジは使い放題で、

フライト中は、

機内で仕事もできます。

各国ごとの食事と、仕事をしながら、

少しずつ目的地に向かうみたいな

「お金は無いが、時間はある」スタイルで、

移動しましたが、

こういう経験ができるのも

今だけだと思っています。

早く事業がもっと上手くいって、

さっと移動できるようになりたいものです笑

🔻パリに到着

━━━━━━━━━━━━━━

パリのシャルル・ド・ゴール空港に到着。

そこからバスとメトロでホテルへ。

すでにフランス時間は18時であり、

軽く散策して終わり。

![IMG_8780.jpeg](202%20%E3%83%95%E3%83%A9%E3%82%A4%E3%83%88%E4%B8%89%E6%98%A7%E2%9C%88%EF%B8%8F/IMG_8780.jpeg)

約1週間ほど、パリ滞在になります。

さて、どんなことか起きるのか？

今から楽しみです！
