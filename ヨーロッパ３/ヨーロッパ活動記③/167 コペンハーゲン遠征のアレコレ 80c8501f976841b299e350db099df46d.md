# 167:コペンハーゲン遠征のアレコレ

2025/02/03

今日はデンマークを出発し、

スウェーデン・ストックホルムへの移動日。

コペンハーゲンでの滞在を振り返りながら、

次の目的地での活動をより良いものにするための準備を整えました。

━━━━━━━━━━━━━━

ストックホルムには **18時到着予定のフライト** だったため、本格的な視察は明日から。そのため、今日は **6時間前にコペンハーゲン空港に到着** し、ラウンジで過ごすことにしました。

僕は **プライオリティパス** を保有しているので、早めに空港のラウンジへ。コペンハーゲンのラウンジでは、なんと **カールスバーグのビールサーバーが設置されており、飲み放題** というスタイル。さすがデンマーク。

しかし、その一方で **食事のバリエーションは少なく**、ガッツリとした料理はあまりありませんでした。

特に、ダブリン空港のラウンジと比べると、その差は歴然。あちらは料理の種類が豊富で、温かい食事も充実しているので、コペンハーゲンのラウンジはちょっと寂しく感じました。

とはいえ、ラウンジでの時間はほぼ仕事に充てていたため、あっという間に過ぎていきました。こうして、約 **1週間のコペンハーゲン滞在** もこれで終了。

# **🔻コペンハーゲンで得たもの、得られなかったもの**

今回のコペンハーゲン滞在では、**僕自身のデザインやアートの感覚が広がり、FOMUSの枡へのアイデアも複数生まれました。**

特に、

✔ **デザインミュージアムでの視察** から、伝統工芸をアートとして世界に発信する方法のヒントを得たこと。

✔ **HAY Houseやロイヤルコペンハーゲン本店での学び** を通じて、FOMUSのパッケージデザインの方向性を考えられたこと。

✔ **枡をアート作品として展示する可能性** を考え、佐木島のギャラリーでの活用方法を模索できたこと。

など、多くのインスピレーションを得ることができました。

しかし、一方で **現地の人や企業とのコネクションを作ることはできませんでした。**

滞在前には **五つ星ホテルにもメールでアプローチ** しましたが、残念ながら **1通も返信なし**。まあ、これは想定内ですが、やはり「コンテンツを持っていくだけでは不十分」だということを痛感しました。

**現地に住み、じっくりと関係を築く時間がなければ、ビジネスの繋がりを作るのは難しい。**

この部分は、今後のFOMUSの海外展開を考える上で大きな課題になります。とはいえ、今回で得られたものも多かったので、引き続きアプローチを続け、コネクションを作る努力を続けていきます。

# **🔻次回のコペンハーゲン訪問に向けて**

今回、現地との繋がりを作ることはできませんでしたが、**5月以降に再びコペンハーゲンを訪れ、枡フォト撮影を実施予定。**

その際には、

✔ **現地のモデルやクリエイターとのコラボを積極的に進める。**

✔ **より時間をかけて、現地の文化やビジネス関係者と交流を深める。**

✔ **今回得たインスピレーションを元に、FOMUSのアートプロジェクトをさらに強化する。**

など、より踏み込んだ活動をしていきたいと思います。

# **🔻ストックホルムでの活動へ**

さて、ここからは **スウェーデン・ストックホルム編。**

今回の滞在期間は限られており、**自由に活動できるのは3日間ほど**。

そのため、短期間でどれだけFOMUSブランドの開拓に活かせる情報を得られるかが重要になります。

ストックホルムでは、

✔ **スウェーデンのデザイン市場を調査し、FOMUSが入り込めるポイントを探る。**

✔ **現地のクラフトやアートギャラリーを視察し、枡とのコラボの可能性を探る。**

✔ **北欧ブランドのビジネスモデルを研究し、FOMUSの方向性に活かす。**

などを重点的に行う予定。

特に、スウェーデンのデザインは **「シンプルかつ機能的」** なものが多く、日本の伝統工芸とも相性が良いはず。そのため、ここでの視察がFOMUSの展開に大きなヒントを与えてくれるかもしれません。

# **🔻デンマーク滞在を振り返って**

今回のコペンハーゲン滞在では、デザインやアートのインスピレーションを得るという面では大成功でした。しかし、ビジネス的な成果としては、まだまだ課題が多いことも明確になりました。

✔ **得られたもの**

・ **FOMUSの枡のデザインやパッケージに関する新たなアイデア**

・ **佐木島ギャラリーでの展示方法のヒント**

・ **デンマークのデザイン文化を学び、FOMUSのブランド強化に活かす視点**

✔ **得られなかったもの**

・ **現地の人や企業との直接的なコネクション**

・ **五つ星ホテルやデザインショップとのビジネス関係の構築**

この経験を踏まえて、今後の海外展開では **「滞在期間の確保」や「現地での交流機会の増加」** を意識しながら動いていきます。
