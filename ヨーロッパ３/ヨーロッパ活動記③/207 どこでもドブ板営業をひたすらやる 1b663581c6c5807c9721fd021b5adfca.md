# 207:どこでもドブ板営業をひたすらやる

2025/03/14

今日は午後から、ミーティング。

1月にリスボン
🇵🇹で、たまたま出会った、フランスに住んでいる日本人の方に紹介してもらった

パリ在住でホテルのリッツでフレンチのサービスをしている方とのミーティング。

僕のミッションは、

FOMUSの枡を世界で展開させることであり、

枡を使ってもらえる可能性があるなら、

どこへでも行きます。

今回も、その方と、その方との繋がりで、

日本酒の重鎮の方がいるらしく、

会える予定でしたが、

スケジュール的に厳しく

今回は会えなかったのですが、

フランスでのガストロノミーとしての枡展開に繋がりそうな話を聞くことができました。

すごく上手くいければ、

パリの日本酒コンクールで、

FOMUSの枡が使われるかもしれません。

まだ最初の入り口段階であり、

大きなことが決まったわけではありませんが、

今度の5月に、

日本酒の大きなイベントがあるらしく、

僕もどんな形でもいいので、

関わりたいという話をさせていただきました。

パリでの繋がりが、たくさんできる可能性があり

仕事でなくてもいいので、

人脈作りとしては、最高の場所です。

なので、どうにか参加したいところ。

この辺りの話を、今回はさせていただきました。

パリ2日目の、アート界隈へのアクションも、

今回の日本酒関連のアクションも、

FOMUSがこれまでに、

枡を活用して、アートもお酒もコラボをしてきたからこそ、このような機会に結びつけることができたので、

日本で色々とやってきたことは、

全く無駄では無いし、その経験や活動が、

今に繋がっています。

また、僕が昔にフレンチレストランのサービスをやっていたこともあり、

相手からしても、親近感がわくそうです笑

また、フランスでホテルマンをやっている人は、

大体途中で独立するらしく、

お酒のインポーターとして

活動する人が多いそうです。

つまり、日本酒を扱う人も多い。

逆にいうと、そこで競合になるわけで、

僕は引き続き、お酒の器である「枡」で、

日本酒を提供する側のサポートとして、

争わず、1番強いところと

コラボができたらと考えています。

そのためにも、

日本酒の重鎮と出会うことは、

重要です。

今回は残念ながら会えませんでしたが、

アイルランドとフランスは近いので、

またすぐに飛ぼうと思いますし、

5月の下旬に開催されるイベントにも、

ぜひ参加したいと考えています。

これも、実際に足を運んだからこそ、

このような話をしてもらえたわけで、

改めて、知らない土地でも、

ドブ板営業をすることの大切さを感じでいます。

また、その後は、

前に教えてもらった

日本の伝統工芸のお店に顔を出して、

「木の器」が足りていないことを聞かせてもらいましたし、枡の話も面白がってもらいました。

こういう出会いも、大切だなぁと。

人が人を紹介し繋いでいく文化は、

パリではしっかりとできていて面白いです。

ダブリンと比べても、

日本人が圧倒的に多いし、

こっちでビジネスをしてる人も多く、

日本人の村〈おそらくたくさんある〉に入り、

そこから、フランス村に入れそうな流れがあります。

逆に、アイルランドのダブリンは、

日本人もそこまで多くなく、

さらに経営者が少なく、アイルランド人の経営者と繋がってる人も少ないため、

いまは、難航しています。

国によって、集まる人も違うので、

この辺りは、とても面白いし、

実際に来なかったら分からないことなので、

実際に足を運んで情報収集したり、ドブ板営業するのは、めちゃくちゃ大切だと感じています。

明日には、アイルランドへ戻りますが、

今回のパリ遠征もたくさんの収穫がありました。

詳しくは明日の記事でまとめたいと思います！
