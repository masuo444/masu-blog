# 75:日本からセブ島に荷物を送る際の注意点

2024/04/16

どうも、FOMUSのまっすーです！

今日は「日本からセブ島に荷物を送る際の注意点」

というテーマで話していきます。

▼海外と日本は違う
━━━━━━━━━━━━━━
今回は、僕の実体験をもとに、
日本からセブ島へ荷物を送ることの注意点をご紹介したいと思います。

この記事を書けるのは、
僕が日本からセブ島へ荷物を送ったからです。

話は遡ること1ヶ月前。

僕のコンタクトレンズの数が

少なくなってきたことと、

薬のパブロンを2月で

全て使ってしまったために、
日本からセブ島へ、

コンタクトとパブロンを郵送しました。

利用したのは、最速で届くという「EMS」

僕も台湾に何度か枡や写真集を送ったことがあるのですが、とても早く荷物が届きます。

同じ要領で、
僕はセブ島の語学学校宛に、僕の名前を書いて、発送しました。
※これはオンラインで住所などを入力します
※手書きはありません

僕の送った場所から数日後には、
東京の国際郵便局へ届けられ、
フィリピンへと発送されました。

日本からフィリピンに荷物を送るとなると、
まずは、マニラへ送られます。

その後、マニラから他の島［今回はセブ島］に
送られていくという流れになります。

そして、

セブ島の大きな郵便局に荷物が送られてて、
近くの小さな郵便局へと発送されて、
そこから、住所の記載された場所に送られるということです。

これが一般的な流れになります。

しかしながら、
**僕の場合、受け取るのに1ヶ月以上掛かりました。**

こちらが履歴になります。

日本の追跡番号で調べられる限度

フィリピン到着後はフィリピンのサイトで確認

日本ではスムーズに事が進んでいるのですが。
フィリピンになってからはとにかく遅い。

さらに、これは後から知ったのですが、
追跡番号で確認できるのは、

フィリピンに着いて、
その後のセブ島に発送されたところまで。

それ以上は情報が出てこないんです。

僕の場合、4月5日に荷物が移動されたというところまでは分かっていて、その後の情報がない。

さらに、セブ島では、

基本的に荷物を取りに

行かなければならないとのこと。

大きな荷物でなければ、
届けてもらえるらしいのですが、
なぜか、待っても届かない。。。

そして、痺れを切らして、
授業中に先生と一緒に郵便局に電話をし、
対応してもらいました。

本来は1人でできればいいのですが、
郵便局の人は現地の言語で話すので、
僕には難し過ぎました。

そして、郵便局に、僕の荷物がどこにあるか確認してもらい、その日のうちにピックアップ。

※2階の小さい空間に郵便局が！

ここまで1ヶ月と10日！
約5週間も掛かってしまいました。

この期間中は、

コンタクトではなく、メガネで過ごし、
薬もないので、

風邪には特に気をつけて生活をしました。

というわけで、
日本からセブ島に

荷物を送るには注意が必要です。

ま、そもそもの話、

僕が3ヶ月分のコンタクトレンズと、
薬を持ち込んでいれば

郵送する必要も無かったわけで、
事前の準備は大切だなぁということを学びました。

この記事を読んでから海外へ行く方は、
ぜひ足りなくならないように、

早いうちから準備をする事をオススメします！

てはまた！
