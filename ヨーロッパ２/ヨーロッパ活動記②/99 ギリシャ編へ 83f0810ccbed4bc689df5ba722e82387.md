# 99:ギリシャ編へ

2024/11/10

さて、ギリシャに到着しました！

マルタからだと、約1時間半で着くので、

到着時刻が0時30分！

しかも入国審査なしで、

入国できました。

この時間に到着して、

空港からアテネ市内に行くのは、

約1時間ほどかかるし、

治安もあまりいいわけではないことを

事前に知っていたので、

今日は空港宿泊！

ホテルではありません。

リアルの空港宿泊です。

というわけで、

寝れるところを探します。

一階のイスは全て埋まっており

寝れる場所はなし。

2階へ上がってみると、

なんと、こんな窪みのあるギャラリーが！

![IMG_2900.jpeg](99%20%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E7%B7%A8%E3%81%B8/IMG_2900.jpeg)

しかも、すでに数人ここで寝ています。

というわけで、

僕もここで寝てみるチャレンジ！！

空港で、寝るのは4回目ですが、

このように床で寝るのは初めてです。

![IMG_2901.jpeg](99%20%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E7%B7%A8%E3%81%B8/IMG_2901.jpeg)

こんな感じで横になり、

持ってきていた袋に入ったサンダルと、

斜めがけのショルダーバックをうまく枕にして

荷物は全て、壁と僕の間に挟むようにして

完全防備。

すぐには寝れなかったので、

YouTubeを少し見てから就寝。

大体5時間くらいここで、休みました。

そして、朝6時半ごろに、

アテネ市内へ。

調べると

ホテルの近くまで行くバスがあるとのことで、

乗ったのですが、

なぜか途中で全員降ろされるという事件。

仕方なく、そこから地下鉄で、駅を目指します。

当然、お金もその分追加で払ってます。

満員電車の地下鉄で揺られて、

なんとか最寄りへ。

そして、すぐに荷物をホテルへ預けます。

今日一日、予定は何も無かったので、

とりあえずホテルのロビーで、

どこに行くかを調べます。

2日目に完璧な状態で、

ギリシャの世界遺産である

パルテノン神殿には行きたかったので、

結果的に、アテネの考古学博物館へ。

それと同時に、

パルテノン神殿のチケットも

同時に購入しました。

というわけで、

朝8時半からオープンする博物館へ。

![IMG_2904.jpeg](99%20%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E7%B7%A8%E3%81%B8/IMG_2904.jpeg)

さすがにオープンしてすぐなので、

人はあまりいませんでした。

基本的には、

「どういう展示の仕方がいいのか？」

「FOMUSに何か活かせるものはないか？」

ということを考えながら。

歴史とかが好きな人は、

もっと楽しめると思うのですが、

僕があまり得意でなく、

その辺りをもっと興味があれば

より入り込めるのに〜と思いましたが、

残念ながらその感覚は今の僕にはあらず。。

それでも、楽しめました！

その後はお昼探しへ。

ケバブの店を見つけたので、

テイクアウトしようと思ったのですが、

イートインもできるようだったので、

入ってみることに。

僕の知ってるケバブは、

クルクル回ってるのを

削ぎ落として食べるものなのですが、

そういうのではないものが出てきました。

![IMG_2935.jpeg](99%20%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E7%B7%A8%E3%81%B8/IMG_2935.jpeg)

パンと、お肉と焼きトマト、オニオン。

これで約6ユーロ。

ま、いいでしょう笑

料理はちゃんと美味しかったです！

その後は、無料で登れる丘へ。

ここでも、

アテネの景色を一望することができました。

![IMG_2946.jpeg](99%20%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E7%B7%A8%E3%81%B8/IMG_2946.jpeg)

![IMG_2948.jpeg](99%20%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E7%B7%A8%E3%81%B8/IMG_2948.jpeg)

![IMG_2953.jpeg](99%20%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E7%B7%A8%E3%81%B8/IMG_2953.jpeg)

これでも十分綺麗なのですが、

奥に見えるパルテノン神殿からの景色は

また別なものなのでしょう。

明日行く予定なので、

楽しみにしておきます。

その後も、少しアテネを散策して、

ホテルにチェックイン。

正直、マルタ最終日から

寝てないので、疲れも溜まりまくっていて

※ハイキングもしてるし

※前日の朝から次の日の夕方まで活動

一旦、横になることにしたのですが、

結局、そのまま寝てしまい、

ふと目覚めたら朝一時。

この時間に目覚めてもやることないので、

そのまま二度寝。

次の日は、

パルテノン神殿が朝8時から予約してるので

それに間に合えばよいのです。
