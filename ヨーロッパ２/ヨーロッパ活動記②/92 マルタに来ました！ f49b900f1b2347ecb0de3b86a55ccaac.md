# 92:マルタに来ました！

2024/11/03

さて、朝4時からダブリンのラウンジで

くつろぎ、

6時半のフライトで、マルタ共和国へ！🇲🇹

フライトの時間だけはがっつりと寝ました。

※ここしか睡眠をとれる時間が無かった

🔻なぜ、マルタにきたの？

━━━━━━━━━━━━━━

もともとマルタに行く予定は無かったのですが、
アイルランドに到着して最初の週の
ミートアップイベントで出会った
ウクライナの友達が、

マルタに住んでいて、
「来るなら泊まってね！」といっても

言ってもらっていたので、
本当に来てしまいました😁

さらに言うと、

ちょうどアイルランドに家が無いタイミングなので、他の国へ行くのも、ベストなんです。

※家借りてたらいない日も支払う必要あるし

というわけで、

このタイミングでマルタに来ました。

🔻また居候！笑

━━━━━━━━━━━━━━

空港着いたら、すぐに友人の家へ。

乗り継ぎがあるとのことだったのですが、

一つ目は空港からだったので、

分かりやすくて乗れたのですが、

途中での乗り換えでは、

バスが時間通りに来なかったので、

約20分ほど、

重たい荷物を持ったまま徒歩で移動、

もう慣れっこなので、この程度は問題なし！

そして、家に到着して、

久しぶりの再会！

年齢的には10才ほど離れていますが、

そんなの関係なく僕は泊まります笑

到着して、

すぐにシャワーを借りてリフレッシュ！

その間に、彼が、

照り焼きチキンを作ってくれていました。

彼はよく日本人の学生を家に呼んで、

日本料理を作ることをやってるので、

色々と詳しいんです。

その後は、
マルタの街を案内してもらいました。

今回は、スリーマという場所に家があるので、

歩いて20分ほどのセントジュリアンへ。

マルタの景色が一望できる場所へ

連れてってくれました。

そして、ディナーは、ビーチ近くの場所で。

マルタのビールも飲みました。

さすがに

睡眠不足だったので、

12時前には就寝。

というわけで、

ここからマルタ編に突入です！

果たしてどんな出会いがあるのか？

楽しみです！
